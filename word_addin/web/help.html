FDA Oncology Copilot - Help

Getting Started

FDA Oncology Copilot helps you generate FDA-style executive summaries for oncology documents. This add-in uses Retrieval-Augmented Generation (RAG) to create summaries that follow FDA regulatory language conventions.

How to Use

Method 1: Summarize Selected Text

  1. Select text in your Word document that you want to summarize
  2. Open the FDA Oncology Copilot add-in from the Insert tab
  3. Select a data source from the dropdown menu
  4. Apply filters if needed (disease type, trial phase, document section)
  5. Click "Summarize Selected Text"
  6. Review the generated summary and reference documents
  7. Click "Insert Summary into Document" to add the summary to your document

Method 2: Upload a Document

  1. Open the FDA Oncology Copilot add-in from the Insert tab
  2. Select a data source from the dropdown menu
  3. Scroll down to the "Upload a Document" section
  4. Click "Choose File" and select a document from your computer
  5. Click "Upload and Process"
  6. The extracted text will be automatically inserted at your cursor position
  7. You can then select this text and use Method 1 to summarize it

Data Sources and Filters

Data Source: Select the knowledge base to use for generating summaries. Different sources may contain different types of FDA documents.

Disease Type: Filter reference documents by disease type (e.g., NSCLC, breast cancer).

Trial Phase: Filter reference documents by clinical trial phase (e.g., Phase 1, Phase 3).

Document Section: Filter reference documents by document section (e.g., efficacy, safety).

Supported File Formats

The add-in supports uploading and processing various file formats, including:

  • Text formats: .txt
  • Office documents: .docx, .pdf
  • Spreadsheets: .csv, .xlsx
  • Web formats: .html, .xml, .json
  • Markdown: .md

When you upload a file, the add-in extracts the text content and inserts it into your document. You can then select this text and generate a summary using the standard summarization process.

Troubleshooting

  • No data sources available: Make sure the backend service is running and vector indices are built.
  • Error generating summary: Check that your OpenAI API key is valid and has sufficient credits.
  • Add-in not responding: Try refreshing the add-in or restarting Word.
  • Text not selected: Make sure you've selected text in your document before clicking "Summarize".
  • File upload errors: Ensure your file is in a supported format and is not corrupted.
  • Large files: Very large files may take longer to process or may be too large for the system.

About

FDA Oncology Copilot is designed to help regulatory writers and medical professionals create FDA-style executive summaries for oncology documents. It uses advanced AI techniques to ensure summaries maintain precise medical terminology, focus on statistical significance and clinical relevance, and follow standard FDA regulatory language conventions.
